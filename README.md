# King Cholao - Sistema de Analítica Financiera 2026

Este repositorio contiene la arquitectura de datos para la gestión y automatización financiera del establecimiento **King Cholao**. El objetivo es transformar los registros operativos diarios en reportes ejecutivos precisos utilizando Python y Google Colab.

## 🛠 Estructura del Proyecto
El proyecto está organizado para garantizar la integridad y trazabilidad de la información:

```text
/
├── data_raw/          # Documentos originales (fuente de verdad)
├── data_processed/    # Resultados consolidados y analizados
└── scripts/           # Lógica de procesamiento en Python

git pull origin main

git add .
git commit -m "Actualizando reportes financieros"
